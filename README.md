# grooming-saloon-site
ГавМяу — космический сайт груминг-салона с неоновым дизайном. Включает каталог услуг, профили мастеров и отзывы. Есть умная форма записи: выбор даты, времени (9–21 ч) и услуги. Занятые слоты скрываются, номера телефонов валидируются. Данные хранятся локально. Сайт адаптивен, работает без сервера. 
 1. Особенности:
а) Уникальный дизайн - космическая тематика с неоновыми акцентами (чёрный, синий, красный, фиолетовый)
б) Полная адаптивность - корректно отображается на всех устройствах
в) Плавные анимации - переходы между вкладками и интерактивные элементы
г) Система записи - календарь с выбором даты и времени
д) Валидация форм - проверка российских номеров телефонов (11 цифр)
е) Локальное хранение - запись данных в localStorage
  2. Функционал
а) Вкладка «Услуги»
+6 видов услуг с описанием и ценами
+Кнопка записи на услугу
б) Вкладка «Грумеры»
+Карточки мастеров с фото
+Информация об опыте и образовании
+Специализация каждого мастера
в) Вкладка «Отзывы»
+Реалистичные отзывы клиентов
+Рейтинг (5 звёзд)
+Дата посещения
  3. Форма записи
+ Ввод ФИО
+ Российский номер телефона (маска +7)
+ Выбор даты через календарь
+ Выбор времени (9:00 - 21:00, интервал 1 час)
+ Выбор одной услуги из списка
+ Подтверждение записи
+ Блокировка занятых временных слотов
